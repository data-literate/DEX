# Support

This document explains how to get help for DataEngineX (DEX).

## Where to Ask

- **Questions / usage help**: GitHub Discussions
- **Bug reports**: GitHub Issues (Bug template)
- **Feature proposals**: GitHub Issues (Feature template)
- **Security concerns**: Follow `SECURITY.md`

## Before Opening an Issue

Please check:

1. `README.md`
2. `docs/README.md`
3. Existing Issues and Discussions

## What to Include in Help Requests

- Environment details (OS, Python version, local/cloud path)
- Steps already tried
- Error logs (sanitized)
- Minimal reproducible example if possible

## Community and Maintainer Expectations

- Be respectful and specific.
- Keep requests scoped to one problem per issue.
- Maintainers review as time permits (small-maintainer project).

## Official vs Unofficial Forks

Only this repository is the official DataEngineX project.
Forks are welcome, but support for forks is provided by fork maintainers unless explicitly stated.
