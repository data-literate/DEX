# Learning & Examples

Python learning examples demonstrating various concepts and patterns.

## Files

- `async_http_client.py` - Async HTTP client patterns with FastAPI
- `context_managers.py` - Context manager patterns with database connections
- `strict_decorators.py` - Decorator patterns and strict validation
- `var_len_args.py` - Variable-length arguments (*args, **kwargs)
- `yield_keyword.py` - Generator and yield patterns

## Purpose

These files demonstrate Python concepts and can be used for:
- Learning and reference
- Proof-of-concepts
- Quick prototyping

They are not part of the main application code.
