Thank you for contributing to DEX.

Guidelines
- Create small, focused PRs.
- Run linters and tests locally before pushing.
- Include a clear PR description and reference issue numbers.
- Use semantic commit messages (e.g., `feat:`, `fix:`, `chore:`).

Local checks
- `uv lock`
- `uv sync`
- `uv run poe lint`
- `uv run poe test`

Code review
- Assign reviewers via PR and wait for required approvals before merging.
